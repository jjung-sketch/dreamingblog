🔗 GitHub 연결

파일 저장/삭제를 위해 GitHub Personal Access Token이 필요합니다.
토큰은 브라우저 세션에만 저장되며 탭을 닫으면 사라집니다.

토큰 발급 방법

  1. GitHub 로그인 → 우측 상단 프로필 → Settings
  2. 좌측 맨 아래 Developer settings 클릭
  3. Personal access tokens → Tokens (classic)
  4. Generate new token (classic) 클릭
  5. Note에 아무 이름, Expiration: No expiration, repo 체크
  6. 맨 아래 Generate token → 토큰 복사

대시보드 블로그 소개

사이트 소개

WC2026 Hub는 2026 FIFA 월드컵을 더 쉽고 빠르게 즐기기 위해 만든 팬 대시보드입니다. 복잡한 여러 사이트를 돌아다닐 필요 없이, 실시간 순위표·경기 결과·남은 일정을 한 화면에서 확인할 수 있습니다.

주요 기능

  • 📊 그룹 순위 — A~L 12개 그룹 전체 승점·득실·골득실 자동 계산
  • ✅ 경기 결과 — 완료된 경기 스코어를 날짜별로 모아보기
  • 📅 남은 일정 — KST(한국 시간) 기준으로 언제, 누구 vs 누구인지 한눈에
  • 📝 분석 블로그 — 주요 경기 결과 분석 및 전망 기사 (한국어·영어)

데이터 출처

경기 데이터는 오픈소스 프로젝트 openfootball의 공개 데이터를 사용합니다. 본 사이트는 FIFA와 무관한 팬 프로젝트이며, FIFA의 공식 후원·인증을 받지 않았습니다.

업데이트 주기

대시보드는 페이지를 열 때마다 최신 데이터를 자동으로 불러오며, 5분마다 자동 새로고침됩니다. 블로그 기사는 주요 경기 후 수시로 업데이트됩니다.

면책 조항

본 사이트는 정보 제공 목적으로 운영되는 비공식 팬 사이트입니다. 데이터의 정확성을 위해 노력하나, 실시간 공식 정보는 FIFA 공식 채널을 통해 확인하시기 바랍니다.

문의

피드백이나 오류 신고는 언제든지 환영합니다.
